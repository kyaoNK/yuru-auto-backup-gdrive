# yuru-auto-backup-gdrive

Premiere Pro のプロジェクトファイル (`.prproj`) を、毎日 1 回の定時ジョブで Google Drive 同期フォルダにコピーする Windows 向けデスクトップアプリ。実際のクラウドアップロードは Google Drive for desktop に任せる、軽量な常駐型ツールです。

既存の PowerShell スクリプト運用（メモ帳でパス書き換え → 右クリック実行）を、設定 GUI と常駐スケジューラに置き換えます。

## できること

- 監視元フォルダを再帰スキャンし、`.prproj` を Google Drive 同期フォルダに `<元の名前>_Latest.prproj` でコピー（平置き、上書き）
- 毎日 1 回の定時実行（既定 09:00、UI で変更可）
- PC がオフ／スリープで定時を逃した場合、次回起動時に取りこぼしを実行
- Google Drive for desktop が起動しきっていない場合、出力先が見えるまで最大 5 分ポーリング
- システムトレイ常駐（左クリックで設定画面、メニューから「今すぐ実行」「終了」）
- Windows ログオン時に自動起動（ON/OFF 可）
- ログ画面でレベル別（情報 / 警告 / エラー）にフィルタ表示

## バックアップの対象条件

以下を**すべて**満たすファイルだけがコピーされます。

- 拡張子が `.prproj`
- 祖先フォルダのどこかが `^\d{6}\(` にマッチ（例: `250304(3)_クイズ`）
- パスのどこにも `Auto-Save` を含まない
- 設定で指定した「除外フォルダ（パス）」のサブツリー外
- 設定で指定した「除外フォルダ名（パターン）」に祖先名が一致しない（大文字小文字区別なし）

最初の 3 つはハードコード固定、後ろ 2 つは UI から自由に編集できます。

## インストール

### ビルド済みバイナリ

リリースページから `.msi` または `.exe` を取得してインストールしてください（※ リリースは追って公開）。

### ソースからビルド

必要なツール:

- [Rust](https://www.rust-lang.org/tools/install) (1.85+)
- [Node.js](https://nodejs.org/) (20+)

```bash
git clone https://github.com/<your-account>/yuru-auto-backup-gdrive.git
cd yuru-auto-backup-gdrive
npm install
npm run tauri build
```

ビルド後の MSI/EXE は `src-tauri/target/release/bundle/` 配下に生成されます。

### 設定とログの保存先

実行ファイルと同じディレクトリに `data/` フォルダを作成して `config.json` と `logs/backup.log` を置きます（ポータブル運用）。`Program Files` 配下など書き込み不可な場所にインストールされている場合は、自動的に `%USERPROFILE%\yuru-auto-backup-gdrive\` にフォールバックします。

## 使い方

### 1. 初回起動

アプリを起動するとダッシュボードが開きます。最初は監視元・出力先が未設定なので「設定」タブから設定してください。

### 2. 設定

**監視元フォルダ**: Premiere Pro のプロジェクトを置いているフォルダ（例: `F:\pedantic制作`）。

**出力先フォルダ**: Google Drive 同期配下の任意のフォルダ。「Google Drive を検出」ボタンを押すと、レジストリ／ドライブレター／慣習パスを総当たりで検出し、候補一覧を表示します。候補をクリックするとそのルートを起点にフォルダ選択ダイアログが開きます。検出に失敗した場合は「選択…」ボタンから手動で選んでください。

**実行時刻**: HH:MM 形式（既定 09:00）。

**自動起動**: Windows ログオン時に常駐させる場合は ON。

**除外フォルダ（パス指定）**: 監視元の中で、特定のサブフォルダ配下を丸ごと除外したいときに追加。サブツリー全体が除外されます。

**除外フォルダ名（パターン）**: `Cache`, `Proxy`, `Render` のように、フォルダ名で除外したいときに追加（大文字小文字区別なし）。祖先のどこかにこの名前のフォルダがあれば、そのファイルは対象外になります。

最後に「保存」を押すと、設定がディスクに永続化され、スケジューラが新しい設定で再武装されます。

### 3. ダッシュボード

- 次回実行時刻、最終実行時刻、最後のサマリ（コピー件数 / エラー件数）が表示されます。
- 「今すぐ実行」ボタンで定時を待たずに手動実行できます。
- ジョブ完了時はトーストで結果が通知されます。

### 4. トレイ

ウィンドウを閉じてもアプリは常駐します。タスクトレイのアイコンから:

- 左クリック: 設定画面を表示
- 右クリック: メニュー（設定を開く / 今すぐ実行 / 終了）

完全に終了したい場合はトレイメニューの「終了」を使ってください。

### 5. ログ

「ログ」タブで `backup.log` の直近 500 行を表示。情報 / 警告 / エラーでフィルタできます。詳細を読みたい場合は設定タブの「設定フォルダを開く」からエクスプローラで `logs/backup.log` を開けます。
